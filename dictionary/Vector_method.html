vector method

* create
* iterator
* add
    * push_back: Push the elements into from the back
    * insert: Insert new elements before the element at the specified position.
* change
    * swap: Swap the contents of one container with another container of same type. Sizes may differ.
    * assign: Assign new value to the container by replacing old ones
    * capacity: Capacity management, including resize, shrink, and reserve.
* query
    * at: Return the element at index 
    * front: Return the first element
    * back: Return the last element
    * size: Return the number of elements
    * max_size: Return the maximum number of elements
    * capacity: Return the size of the storage space currently allocated
    * empty: Return whether the container is empty.
* delete 
    * pop_back: Pop or remove elements from the back
    * erase: Remove elements from a container from the specified position or range.
    * clear: Remove all the elements of the vector container
